# Eric Roche API Resume
## Description
You can fetch my resume as a REST API
## Technology
This is a REST API that accesses a PostgreSQL hosted on Supabase. Front-end is hosted in Vercel.
-  PostgreSQL
-  Next.js
-  Vercel
-  Tailwind CSS
-  Supabase
-  React
